# 产品需求文档 (PRD) · 杏铃聊天设计画布重设计（极简版）

## 1. 背景与目标

### 1.1 背景
当前 `design/preview.html` 是 Trae Work 风格暗色线框，紫罗兰主色 + 近黑底，落入「AI 通用美学」陷阱。用户要求改为**极简风**，拒绝奶油色与暖色调。

### 1.2 目标
重新设计单一 HTML 设计画布：
1. **极简美学**：黑白灰为主，单点强调，大量留白
2. **品牌锚点**：保留「杏」字水印作为唯一签名时刻
3. **保留信息架构**：侧边栏 + 对话 + 设置三段式布局
4. **可下载直链**：单文件 HTML

## 2. 视觉方向（核心约束）

### 2.1 Aesthetic
"Mono Brut"（极简野兽）—— 纯黑底 + 纯白字 + 单色强调 + 极致克制 + 大量负空间

### 2.2 禁止
- 紫色渐变、奶油色、杏色、暖色调
- Inter/Roboto/Arial 系统字
- 白色卡片浮于紫黑底
- 任何渐变背景（除极弱的单色辉光）

### 2.3 鼓励
- 纯黑 #000 / 纯白 #FFF 二色为主
- 极细描边（1px rgba(255,255,255,0.06)）
- 大量负空间
- 衬线展示字 + 几何无衬线正文
- 单一强调色（克制使用）

## 3. 调色板

| Token | 值 | 用途 |
|---|---|---|
| `--ink` | `#000000` | 主背景（纯黑）|
| `--ink-2` | `#0A0A0A` | 侧边栏底 |
| `--ink-3` | `#141414` | 卡片/输入框 |
| `--paper` | `#FFFFFF` | 主文字（纯白）|
| `--paper-2` | `#888888` | 次要文字 |
| `--paper-3` | `#444444` | 三级文字 |
| `--accent` | `#FFFFFF` | 主强调（即纯白，用作主操作）|
| `--signal` | `#7FA67A` | 状态色（流式/在线，唯一的非灰彩色）|
| `--clay` | `#B33A3A` | 危险/停止 |
| `--border` | `rgba(255,255,255,0.06)` | 弱边框 |
| `--border-2` | `rgba(255,255,255,0.12)` | 强边框 |

## 4. 字体

- **展示字**：`Fraunces`（可变衬线，极强个性）
- **正文**：`Manrope`（几何无衬线，干净）
- **等宽**：`JetBrains Mono`
- **水印字**：`Fraunces` 950 字重 + 巨大字号

## 5. 签名时刻

页面右上角放一个巨大的、几乎透明的「杏」字（Fraunces 950），作为唯一装饰元素。承担三重功能：
1. 品牌锚定（杏铃）
2. 视觉签名
3. 暗示「工坊式」手工设计

## 7. 验收标准

- [ ] 单文件 HTML 可直接下载打开预览
- [ ] 至少 8 种控件视觉示范齐全
- [ ] 调色板符合规范，无紫色、无奶油色、无暖色
- [ ] 字体使用 Fraunces + Manrope + JetBrains Mono
- [ ] 包含「杏」字水印签名时刻
- [ ] 移动端竖屏 ≥ 360px 可读
- [ ] 体积 < 80KB
